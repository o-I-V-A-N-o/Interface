# Interface

# Обязательные требования
#  
# [v] Добавить сцену меню, в которой будет:
# 	[v] Картинка на заднем фоне (взять любую, на Ваш вкус);
# 	[?] Панелька в центре экрана, сверху над панелькой должно быть название игры (придумать любое, на Ваш вкус). Название должно иметь минимум два цвета в тексте и 2 разных шрифта;
# 	[v] Кнопка “Новая игра”, которая запускает игровую сцену;
# 	[v] Кнопка “Выйти”, которая выходит из прототипа или останавливает воспроизведение, если прототип запущен в редакторе;
# 	[v] Слева внизу экрана должно быть небольшое текстовое поле, выводящее текущую версию прототипа (например, “ver: 0.2.1234”).
# [v] Добавить меню-паузу в самой игре:
# 	[v] Чтобы поставить паузу, нужно нажать на кнопку ESC первому игроку;
# 	[v] При нажатии на кнопку вызова меню-паузы, игра останавливается, игроки и шарик не могут двигаться, а на экране появляется панель с меню;
# 	[v] В меню-паузе есть кнопка “Возобновить”, она скрывает меню-паузу, а игра возобновляется;
# 	[v] Кнопка “Перезапустить”, текущий уровень перезапускается заново;
# 	[v] Кнопка “Выйти”, которая выходит из прототипа или останавливает воспроизведение, если прототип запущен в редакторе.

# Ограничения
# 
# [v] Весь текст с изменяющимся шрифтом и цветом должен быть реализован через обогащенный текст.
# [v] В главном меню должна быть минимум одна анимация интерфейса.
# [v] Количество жизней игроков должно выводиться на экран, а не в консоль в одном из углов экрана (данная информация должна выводиться на экраны обоих игроков), при этом элемент интерфейса здоровья должен:
# 	[v] Состоять из картинки сердечка (взять любое изображение из интернета);
# 	[v] Иметь текстовое поле, отображающее количество оставшихся жизней;
# 	[v] Быть префабом. То есть у обоих игроков интерфейс одинаковый и взят из префаба.

# Дополнительные требования
# 
# [v] В главном меню и в меню-паузе добавить кнопку “Настройки”, которая:
# 	[v] Открывает вместо панели меню, меню настройки;
# 	[v] Содержит галочку-заглушку с текстовым описанием “Отключить звук”;
# 	[v] Содержать горизонтальный слайдер-заглушку, с припиской “Громкость” сверху и цифрами 0 и 100 слева и справа, соответственно;
# 	[v] Раздел “Сложность”, в котором размещена группа галочек-заглушек с приписками “Легкий”, “Средний” и “Тяжелый”;
# 	[v] Кнопка “Назад”, которая закрывает меню настроек и открывает панель главного меню/меню-паузы.
# [v] Все изменения настроек должны сохраняться между сессиями, то есть после перезапуска игры, параметры должны остаться теми, что были установлены в прошлую сессию.
# [v] Добавить анимацию панелей главного меню, меню-паузы и меню настроек:
# 	[v] При открытии меню, оно должно увеличиваться до базовых размеров из центра экрана;
# 	[v] При закрытии меню, оно должно уменьшаться до нулевых размеров в центр экрана;
# 	[v] Анимация открытия и закрытия должны срабатывать друг за дружкой, то есть при открытии настроек происходит следующее: полностью отыгрывается анимация закрытия меню, полностью отыгрывается анимация открытия настроек.
